# Online Examination System

## Overview
The **Online Examination System** is a web-based application that allows students to take exams online. It provides a secure and convenient platform for conducting exams, evaluating answers, and generating results automatically. The system is designed for educational institutions and training centers.

---

## Features
- User-friendly interface for students and administrators
- Admin can create exams, questions, and manage users
- Supports multiple-choice questions (MCQs)
- Automatic grading and result generation
- Track student progress and scores
- Secure login and user authentication

---

## Technologies Used
- **Backend:** Java, Spring Boot
- **Frontend:** HTML, CSS, JavaScript (or Thymeleaf if using Spring templates)
- **Database:** MySQL
- **Build Tool:** Maven
- **Version Control:** Git & GitHub
